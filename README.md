# Avaliação de consumo de nutrientes

## Descrição
Usa um arquivo fonte contendo as refeições, e outro arquivo com os dados da Tabela TACO, para calcular os nutrientes oferecidos em cada refeição.  
A partir dos dados do peso do paciente, calcula a necessidade calórica e proteica.  
Informando a quantidade aproximada que o paciente ingeriu de cada refeição, é possível calcular a ingestão total e dimensionar um eventual suplemento.  
Ainda oferece uma busca de refeições cadastradas que contenham um determinado alimento procurado
